Projects

Clock 'App'

Purpose Have a convenient way to tell time and select between a variety of features to measure time.

Project Description This project uses command line arguments to simulate a clock app. Based on user input, the Simulator will display between a Simple Clock that displays the time and date (h:m:s m/d/y) every second, a Timer where the user can select the amount of time in hours, minutes, and seconds until it rings, as well as the sound it plays when it rings. Finally, the user can also choose an alarm where they will be able to select what time it rings at and with what sound.

Project design Our project is broken up into Alarm, Alarm Clock, Clock, Standard Clock, Sound, Time, Timer, and Simulator. The Clock class serves a basis for Standard Clock, Timer, and Alarm which will all share the same features of Clock but each will have their own implementation.

To use our project, run the Simulator class with the correct command line arguments. Our Simulator class chooses which feature to employ based on the first argument provided. To use the Standard clock, run the Simulator class with the single argument: standard. To run the Timer, run the simulator class with the arguments, timer . Finally, to run the alarm, run the simulator class with the commands: alarm . The sounds one can choose from are: rooster, bird, marimba, and classic.
